тестирование проводилось в браузере Firefox 49.0.2

*создание новой заметки:
1. нажать кнопку "new note";
2. ввести текст в поле "enter note text", после ввода слова с хештегом, обязательно поставить пробел после него ;
3. нажать кнопку "save".

*отображение списка заметок и тегов:
1. нажать кнопку "all notes".

*просмотр заметки:
1. кликнуть по заметке из списка.

*редактирование заметки:
1. войти в режим просмотра заметки;
2. сделать двойной клик по тексту заметки.

*удаление заметки:
1. войти в режим просмотра заметки;
2. нажать кнопку "delete".

*поиск по хештегам:
1. ввести текст в поле "enter tag for search";
2. нажать клавишу Enter.

*удаление хештега:
1. двойной клик по хештегу из списка справа.
